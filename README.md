# homebrew-tap

Homebrew tap for [UltimaPhoenix](https://github.com/UltimaPhoenix) projects.

## Setup

```bash
brew tap UltimaPhoenix/tap
```

## Available formulae

| Formula | Description | Install |
|---------|-------------|---------|
| [devcoach](https://github.com/UltimaPhoenix/dev-coach) | Progressive technical coaching MCP server for Claude Code and Claude Desktop | `brew install UltimaPhoenix/tap/devcoach` |

## Usage

```bash
brew tap UltimaPhoenix/tap && brew install devcoach
```
